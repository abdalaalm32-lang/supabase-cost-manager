---
name: Shift Expenses and Returns System
description: نظام مصروفات الشيفت ومرتجعات POS مع حماية الدور والطباعة الحرارية
type: feature
---
- **مصروفات الشيفت** (`pos_shift_expenses`): تسجيل مصروفات من درج الكاشير أثناء الشيفت المفتوح
  - مكون `PosShiftExpenses` يظهر فقط عند وجود شيفت مفتوح
  - إضافة/حذف مصروفات مع وصف ومبلغ
  - يتم حساب المصروفات في تقرير إغلاق الشيفت
- **المرتجعات** (`pos_returns` + `pos_return_items`): إرجاع جزئي أو كامل
  - **الحماية**: فقط `مدير` أو `مالك` يمكنهم عمل مرتجع
  - البحث بالفاتورة → اختيار أصناف وكميات → إدخال سبب → تأكيد وطباعة
  - رقم تلقائي بصيغة `RET_YYYY_NNNN`
  - تقرير المرتجعات اليومي متاح لجميع المستخدمين
- **تقرير الشيفت** يشمل: المبيعات + المصروفات + المرتجعات + صافي الشيفت + المتوقع في الدرج
- طباعة عبر iframe بعرض 72mm (80mm حراري)
